achievements

  • Proposed a novel noisy label technique (sentiment suggestive words) for automatically labelling large datasets of tweet.
  • Created a dataset of 1.5 million tweets and labeled it using sentiment suggestive words.
  • Created a few auzilliary datasets usingo ther noisy label techniques.
  • Built a Naive-Bayes classifier and trained it usign different datasets.
  • Deviced a way to select a testset from the training set using entropy based similarity measure.
  • Studies categorical propotional difference (CPD) and minimum feature threshold as the feature selection measure

conclusion

  • Sentiment suggestive words perform well as an automatic labeling technique performs well and at par with other noisy label techniques like emoticons and hashtags.
