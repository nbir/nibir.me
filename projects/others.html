smarter space management system (SSMS)

Smarter Space Management System(SSMS) is an computer aided automation system for a carrying and forwarding agent. Clients can register and book space with the warehouse facility online. Storage transactions are entered and maintained online. Automatic email notifications, and report generation feature for management as well as client is available. An intuitive graphical interface makes the very system user friendly.

language: Java
